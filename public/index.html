Some Things I've Made

Safety Net

The Safety Net is an app that helps people in need connect with those who are willing to help. It was build using EmberJS and AdonisJS with Shaun Willis.

Puppy Adopter

The Puppy Adoption site allows visitors to post new dogs and claim others for adoption. It was built using VueJs and styled with Bulma.

Admin Menu Maker

Menu Maker was made in EmberJs with custom style. In the admin side you can create/read/update/destroy new menu items.

Nasa Picture of the Day

This site uses the Nasa API to display the picture of the day. It's built with VueJS.

Marvel Site

Using marvel's api, I created a app that shows relevant charactors and comics for a specific series. It was built with VueJs.

Horror Soundtrack Maker

This site uses open sourced sound files to create a horror soundtrack. The different keys play overlaping sounds. It's built with vanilla JavaScript.

Twitch Clone

This is a Twitch clone. I used plain js for the grid. It resizes with CSS.

Monster Maker

Monster Maker creates a new monster when you cycle through the different body parts. It was built with VueJS.
